EEPROM24LCXX
A lightweight, easy-to-use Arduino library for interfacing with the 24LCxx family of I2C EEPROMs (e.g., 24LC08, 24LC16, 24LC256, etc.).

- Simple API: Read and write single bytes or write entire strings with ease. (Read Whole strings are W.I.P)

- I2C Protocol: Utilizes the standard Wire.h library for communication.

- Flexible: Designed to work with various memory sizes within the 24LCxx series.

- Lightweight: Minimal memory footprint, perfect for Uno, Nano, and other small microcontrollers.
